## Link safety layer

Some internal concept links point to nodes that are referenced by the graph but are not full pages in the copied sitemap. To prevent dead internal links:

- `scripts/generate-link-aliases.py` crawls `_site` and creates lightweight Eleventy pages under `link-aliases/*.md` for any missing internal path.
- These aliases are excluded from collections and keep every internal href resolvable while fuller node pages are expanded.

## Verification commands

```bash
npm run build
python3 scripts/generate-link-aliases.py
npm run build
```

Then crawl `_site/**/*.html` and verify every same-origin link, image, script, stylesheet, and srcset target resolves locally.
